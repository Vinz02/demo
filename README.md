## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* Java 11
* Maven

### Installation

1. Clone the repo
```sh
git clone https://github.com/Vinz02/demo.git
```
2. Install Maven packages
```sh
mvn clean compile package
```
3. Run the spring boot app
```sh
java -jar demo.jar
```
